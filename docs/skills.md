# OPSI Agent Skills

Installable Agent Skills for using the OPSI CLI from compatible AI agents. Run `opsi agent setup` for automatic host detection and global installation of the complete repertoire. To manage a project-local installation manually, install the repertoire with a compatible Agent Skills installer, or install one focused domain skill and its `opsi-shared` prerequisite.

| Skill | Description |
| --- | --- |
| [opsi](../skills/opsi/SKILL.md) | Use when a Slovenian public-data or OPSI request needs the relevant skill selected. |
| [opsi-shared](../skills/opsi-shared/SKILL.md) | Use when any OPSI CLI skill needs shared installation, output, offline, safety, or error-handling guidance. |
| [opsi-catalogue](../skills/opsi-catalogue/SKILL.md) | Use when discovering Slovenian public-data or OPSI datasets, metadata, resources, schemas, or public pages. |
| [opsi-resources](../skills/opsi-resources/SKILL.md) | Use when inspecting an OPSI resource, its secure access, headers, or bounded preview before the next step. |
| [opsi-download](../skills/opsi-download/SKILL.md) | Use when securely downloading an OPSI dataset or resource and choosing a destination or overwrite handling. |
| [opsi-validation](../skills/opsi-validation/SKILL.md) | Use when checking local or provider data, or OPSI metadata, for integrity issues and remediation. |
| [opsi-analysis](../skills/opsi-analysis/SKILL.md) | Use when querying or converting bounded data, including ZIP, XML, JSON, XLSX, Parquet, or query exports. |
| [opsi-services](../skills/opsi-services/SKILL.md) | Use when Slovenian public data is exposed through WFS and the request needs capabilities, layers, schemas, bounded feature previews, counts, or CSV exports. |
| [opsi-provenance](../skills/opsi-provenance/SKILL.md) | Use when inspecting or verifying OPSI artifact provenance, transformations, or integrity mismatches. |
| [opsi-local-state](../skills/opsi-local-state/SKILL.md) | Use when inspecting or changing the OPSI cache or non-secret configuration. |
| [opsi-diagnostics](../skills/opsi-diagnostics/SKILL.md) | Use when diagnosing OPSI, generating shell completion or Agent Skills, or performing agent setup. |
